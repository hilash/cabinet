# Storefront

This directory demonstrates Cabinet's **linked repo** feature. The `.repo.yaml` file connects this KB page to the Weasleys' Wizard Wheezes storefront website code.

## How It Works

The `.repo.yaml` file tells Cabinet that this page is backed by a local Git repository. The `source` symlink lets you browse the actual website files directly from the sidebar — HTML, CSS, JavaScript, and assets all appear as navigable pages.

When AI agents work on tasks related to [[Marketing]] or product pages, they can read the linked source code for full context. This means an agent updating ad copy can reference the actual storefront markup, or an agent reviewing the product catalog can cross-reference what's live on the site.
